API文档

本工具提供了REST API接口，可以通过程序调用实现文档转换功能。

文档转文本API

POST /api/convert

内容类型: multipart/form-data

请求参数

参数名 类型 必填 描述
file 文件 是 要转换的文档文件

支持的文件类型

  • Word文档 (.doc, .docx)
  • Excel表格 (.xls, .xlsx)
  • PowerPoint演示文稿 (.ppt, .pptx)
  • PDF文档 (.pdf)
  • 文本文件 (.txt)
  • Markdown文件 (.md)

响应格式

成功响应 (HTTP 200):

{
  "text": "转换后的文本内容",
  "filename": "原始文件名",
  "file_size": "文件大小(字节)",
  "processing_time": "处理时间(秒)"
}

错误响应 (HTTP 4xx/5xx):

{
  "error": "错误信息",
  "details": "详细错误信息(如有)"
}

文档转Markdown API

POST /api/convert-to-md

内容类型: multipart/form-data

请求参数

参数名 类型 必填 描述
file 文件 是 要转换的文档文件

支持的文件类型

  • Word文档 (.doc, .docx)
  • Excel表格 (.xls, .xlsx)
  • PowerPoint演示文稿 (.ppt, .pptx)
  • PDF文档 (.pdf)
  • 文本文件 (.txt)
  • Markdown文件 (.md)
  • 音频文件 (.mp3, .wav) - 新增

响应格式

成功响应 (HTTP 200):

{
  "text": "转换后的Markdown内容",
  "filename": "原始文件名",
  "file_size": "文件大小(字节)",
  "processing_time": "处理时间(秒)"
}

错误响应 (HTTP 4xx/5xx):

{
  "error": "错误信息",
  "details": "详细错误信息(如有)"
}

特点

与普通文本转换不同，Markdown转换将保留更多的原始文档结构：

  • 保留标题层级结构
  • 保留列表格式
  • 保留表格结构
  • 保留超链接
  • 对音频文件提取元数据，包括时长、声道数和采样率

示例代码

Python示例 - 转为文本

import requests

url = "http://127.0.0.1:5000/api/convert"
file_path = "example.docx"  # 替换为您的文件路径

with open(file_path, "rb") as f:
    files = {"file": (file_path, f)}
    response = requests.post(url, files=files)

if response.status_code == 200:
    result = response.json()
    print("转换成功!")
    print(f"文本内容: {result['text'][:100]}...")  # 打印前100个字符
    print(f"文件名: {result['filename']}")
    print(f"文件大小: {result['file_size']} 字节")
    print(f"处理时间: {result['processing_time']} 秒")
else:
    print(f"转换失败: {response.json()['error']}")
